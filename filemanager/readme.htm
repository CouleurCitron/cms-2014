About
EFZ FMS : Easy and Simple File management system was built to let you access various files and directories on your website with ease and without requring a ftp connection . I includes standard features like new file , new directory , delete , edit , view and upload files . It was meant to be very simply integrated and used without requring much configuration . The user interface is kept as simple as possible .

Installation

1.To install the script unzip the script in an directory .
2.Make globals.php world writable (777)
3.Log in

Execute the admin.php to access the main interface .
Other settings and control can be accessed admin.php in the script directory .
Admin username : user
Password : pass
